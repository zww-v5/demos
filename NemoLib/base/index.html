说明

基础脚本 base.js 是整个脚本库的基础，其他的脚本都是基于这个脚本来编写。
基础脚本旨在使用原生脚本编写代码，减少对 jQuery 的依赖，借鉴了 http://youmightnotneedjquery.com/ ，只提供了常用的 dom 操作、查找元素等原生脚本没有提供或编写不方便的操作。
兼容的浏览器为：ie9+、chrome、firefox、safari。

与 jQuery 类似，所有的脚本函数都作为 Nemo 或 N 对象的内容，使用时采用 N.getFirstElement(ele) 这种方式调用。

getFirstElement

getFirstElement(selector / element)

获取匹配的元素组中的第一个元素

selector

Type: Selector

一段匹配元素的字符串，与 jQuery 的查找元素的字符串一致，比如 N.getFirstElement('.content h1')

element

Type: Element

一个原生的 DOM 元素，比如 N.getFirstElement(document.querySelectorAll('.content h1'))

                    
                        
                    
                
                    
N.getFirstElement('.list > li');

ready

ready([documentElement,]handler)

在当前文档或指定文档中，当初始的 HTML 文档被完全加载和解析完成之后，执行 handler 函数，而无需等待样式表、图像和子框架的完成加载。

documentElement

Type: Element

指定的文档元素，比如 iframes[i].contentWindow.document

handler

Type: Function

在 DOM 元素完全加载后执行的函数

                    
                        
                    
                
                    
//监听当前文档
N.ready(function(e){
    console.log('加载完了');
});

//监听指定文档
var iframes = document.querySelectorAll(".content-mod-iframe iframe");
N.ready(iframes[i].contentWindow.document, function(e){
    console.log('加载完了');
});

extend

extend([deep,]target,objectA[,objectB...])

描述

deep

Type: Boolean

是否要深度拷贝，默认是浅拷贝。当选择深度拷贝时，如果多个对象的同名属性也是对象时，则该“属性对象”的属性也会进行合并

target

Type: Object

目标对象，后面对象的属性都合并到这个对象中

objectA

Type: Object

用于合并的对象，当前对象的属性值都会合并到目标对象中

                    
var object1 = {
  apple: 0,
  banana: { weight: 52, price: 100 },
  cherry: 97
};
var object2 = {
  banana: { price: 200 },
  durian: 100
};

//浅拷贝，object1 打印出来的内容是：{"apple":0,"banana":{"price":200},"cherry":97,"durian":100}
N.extend( object1, object2 );

//深度拷贝，object1 打印出来的内容是：{"apple":0,"banana":{"weight":52,"price":200},"cherry":97,"durian":100}
N.extend( true, object1, object2 );
